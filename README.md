# DAO - Arc Testnet

A decentralized autonomous organization smart contract deployed on Arc Testnet using Foundry.

## Contract Details

- **Network:** Arc Testnet
- **Contract Address:** 0x3f94600877D990dd966dd83493b454A726A73d95
- **Deployer:** 0x54b4B44749a95070560509B6Ec0be501665CcF63
- **Transaction Hash:** 0xbd460324ebec3854e6489f324588b56b6645a7e349b1d0d5e6b61230b21f2a3b
- **Quorum:** 51%
- **Voting Duration:** 1 day (86400 seconds)

## Features

- Membership management (add/remove members)
- Create proposals with on-chain execution
- Member-only voting system
- Quorum-based proposal finalization
- Admin-controlled proposal execution
- Proposal cancellation by proposer or admin
- Built with Foundry and Solidity ^0.8.30

## Getting Started

```bash
forge install
forge test
forge build
```
